## Descripción

Microservicio REST construido con FastAPI e integración IA usando Groq.

Permite:

- Validar solicitudes
- Clasificar automáticamente
- Asignar prioridad
- Generar justificaciones
- Definir siguiente paso
- Simular integración externa
